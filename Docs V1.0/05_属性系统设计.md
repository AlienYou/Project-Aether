# 05_属性系统设计（Attribute System Design）

版本：v1.0

项目：Project Aether

引擎版本：Unity 2022.3.51f1c1

文档状态：正式版

---

# 1. 文档目标

定义项目统一属性系统。

负责：

* 基础属性
* 成长属性
* 装备属性
* Buff属性
* 最终属性计算

为：

Combat

Skill

Buff

Equipment

提供统一数值来源。

---

# 2. 设计原则

遵循：

数据驱动

统一计算链

支持热更新

支持扩展

支持网络同步

# 35. 结论

Attribute System 是 Project Aether 的统一数值中心。

所有成长、装备、Buff、技能加成最终都通过 Attribute System 计算。

禁止任何系统绕过 Attribute System 直接修改最终战斗数值。
