# 11_网络同步架构设计（Network Architecture Design）

版本：v1.0

项目：Project Aether

引擎版本：Unity 2022.3.51f1c1

网络框架：FishNet

文档状态：正式版

---

# 1. 文档目标

定义 Project Aether 的网络同步架构。

负责：

* 客户端同步
* 服务端同步
* 状态同步
* 战斗同步
* 技能同步
* AI同步
* 数据同步

构建工业级多人动作游戏网络基础设施。

---

# 2. 网络设计目标

支持：

* 单人模式
* 多人联机
* Coop副本
* 世界探索
* Boss战

目标规模：

```text
4人小队

↓

16人场景

↓

后续扩展
```

---

# 3. 网络模型

采用：

```text
Server Authoritative
```

服务器权威架构。

---

# 4. 为什么服务器权威

防止：

* 修改血量
* 修改伤害
* 修改技能CD
* 瞬移
* 加速外挂

---
