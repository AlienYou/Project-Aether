# 01_技术架构设计（Technical Architecture Design）

版本：v1.0

项目：Project Aether

引擎版本：Unity 2022.3.51f1c1

文档状态：正式版

---

# 1. 项目概述

## 1.1 项目目标

Project Aether 是一款工业级网络3D动作RPG。

参考产品：

* 原神
* 鸣潮
* 鬼泣5
* 战双帕弥什
* 绝区零

目标特征：

* 高品质动作战斗
* 开放成长体系
* 多人联机
* 长期运营
* PC优先开发

---

## 1.2 开发原则

遵循：

高内聚

低耦合

模块化

数据驱动

服务器权威

可测试

可扩展

---

# 2. 技术架构总览

```text

↓

Gameplay Layer
（游戏逻辑层）

↓

Domain Layer
（领域层）

↓

Infrastructure Layer
（基础设施层）
```

# 5. 分层设计

## Presentation Layer

负责：

* UI
* 动画
* 特效
* 音频

禁止：

业务逻辑计算

---

## Gameplay Layer

负责：

* 战斗
* 技能
* AI
* 副本

核心业务层。

---

## Domain Layer

负责：

* 属性计算
* 配置解析
* 数据模型

纯业务规则。

---

## Infrastructure Layer

负责：

* 网络
* 文件
* 配置
* 资源

基础设施。

---
